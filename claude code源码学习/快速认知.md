- 管会话状态、桥接和远程连接，让同一个 agent 可以在本地 UI、bridge、direct-connect 之间稳定通信，见 [useReplBridge.tsx](/mnt/e/claude-code-snapshot-backup/src/hooks/useReplBridge.tsx)、[directConnectManager.ts](/mnt/e/claude-code-snapshot-backup/src/server/directConnectManager.ts)、[bridgeMessaging.ts](/mnt/e/claude-code-snapshot-backup/src/bridge/bridgeMessaging.ts)。
