• 要看别人创建的知识库，关键是工作区角色和知识库权限。你被邀请进来后，只有在权限开放给你时才会看到并访问。

  你可以按这个顺序排查/操作：

  1. 确认在正确的工作区
     用左上角工作区切换器进入对方邀请的那个 workspace。(docs.dify.ai (https://docs.dify.ai/zh/use-dify/workspace/readme?utm_source=openai))
  2. 进入 Knowledge（知识库）
     知识库管理页面只对 Owner / Admin / Editor 可见；如果你是 Member，页面通常不可见。(docs.dify.ai (https://docs.dify.ai/versions/3-0-x/en/user-guide/
     knowledge-base/knowledge-and-documents-maintenance/introduction?utm_source=openai))
  3. 让知识库拥有者调整权限
     请 Owner / Admin / Editor 到该知识库 Settings，把权限设置为：

  - “All team members”，或
  - “Partial team members”并把你加入
    否则你看不到该知识库。(docs.dify.ai (https://docs.dify.ai/versions/3-0-x/en/user-guide/knowledge-base/knowledge-and-documents-maintenance/introduction?
    utm_source=openai))

  4. 如果你只是“使用”而不是“管理”
     让同事把知识库绑定到应用，然后给你应用访问权限；Member 角色一般只能使用已发布应用。(docs.dify.ai (https://docs.dify.ai/en/use-dify/workspace/team-
     members-management?utm_source=openai))

  - 你在该工作区的角色是什么（Member / Editor / Admin / Owner）？
