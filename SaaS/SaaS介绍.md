# SaaS (Software as a Service)

即"软件即服务"，是一种软件交付模式：

- 软件托管在云端，用户通过浏览器/客户端访问，无需本地安装
- 按订阅付费（月费/年费），而非一次性购买
- 由服务商负责维护、更新和运维

## 常见例子

Gmail、Slack、Notion、Salesforce、Figma、GitHub

## 与传统软件的区别

| | 传统软件 | SaaS |
|---|---|---|
| 部署 | 本地安装 | 云端访问 |
| 付费 | 一次性买断 | 订阅制 |
| 维护 | 用户自行负责 | 服务商负责 |
| 更新 | 手动升级 | 自动更新 |
