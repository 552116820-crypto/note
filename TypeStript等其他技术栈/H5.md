● H5 页面是国内对使用 HTML5 技术制作的移动端网页的简称。

  本质

  就是一个普通的网页，但主要针对手机浏览器/微信内置浏览器设计，用到了 HTML5 的新特性（如 Canvas、音视频、本地存储、CSS3
  动画等）。

  常见场景

  - 微信里转发的活动页、邀请函、小游戏
  - 品牌营销推广页（翻页动画、抽奖页面）
  - 移动端问卷、表单

  和普通网页的区别

  其实没有本质区别，都是 HTML + CSS + JS。只是"H5"这个叫法强调了：
  1. 移动端优先 — 适配手机屏幕
  2. 用了 HTML5 的新 API — 动画、手势、音视频等
  3. 通常在微信等 App 内打开 — 而不是在桌面浏览器

  注意

  "H5"是中国特有的叫法，国外不这么说。国外直接叫 mobile web page 或 web app。如果你跟外国开发者说"H5"，他们可能不理解。

  简单总结：H5 = 给手机看的网页，技术上就是 HTML/CSS/JS，没有什么特殊的新技术栈。
