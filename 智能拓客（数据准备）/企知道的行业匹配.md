**Assistant:**

我会把它拆成“目标行业库向量化”和“企知道行业向量化”两步来讲，核心就是把每个行业路径转成一串数字向量，再用余弦相似度找最接近的原行业。

做法就是：把两套行业名称都拼成“可表达语义的文本”，然后 embedding，最后算相似度。

你本机现在有 `pandas/openpyxl`，但还没有 `sentence_transformers`。先装：

```bash
pip3 install -U sentence-transformers
```

核心代码可以这样写：

```python
import pandas as pd
import numpy as np
from sentence_transformers import SentenceTransformer

old_file = "/mnt/c/Users/55211/Documents/WXWork/1688854442795280/Cache/File/2026-05/安美行业.xlsx"
qzd_file = "/mnt/c/Users/55211/path/to/企知道数据.xlsx"
out_file = "/mnt/c/Users/55211/path/to/行业匹配结果.xlsx"

# 1. 读取安美原行业
old = pd.read_excel(old_file)

targets = old[old["level"].eq(2)].copy()
targets["target_text"] = (
    "安美行业：" +
    targets["parent_name"].fillna("") +
    " / " +
    targets["name"].fillna("")
)

# 2. 读取企知道行业
qzd = pd.read_excel(qzd_file)

qzd["source_text"] = (
    "企知道行业：" +
    qzd["industry level 1"].fillna("") +
    " / " +
    qzd["industry level 2"].fillna("") +
    " / " +
    qzd["industry level 3"].fillna("")
)

# 3. 中文 embedding 模型
model = SentenceTransformer("BAAI/bge-small-zh-v1.5")

target_vec = model.encode(
    targets["target_text"].tolist(),
    normalize_embeddings=True,
    show_progress_bar=True,
)

source_vec = model.encode(
    qzd["source_text"].tolist(),
    normalize_embeddings=True,
    show_progress_bar=True,
)

# 4. 余弦相似度。因为已 normalize，点积就是 cosine similarity
scores = source_vec @ target_vec.T

# 5. 每条企知道行业取 Top 3 候选
top_k = 3
top_idx = np.argsort(-scores, axis=1)[:, :top_k]

for rank in range(top_k):
    idx = top_idx[:, rank]
    qzd[f"candidate_{rank+1}_score"] = scores[np.arange(len(qzd)), idx]
    qzd[f"candidate_{rank+1}_id"] = targets.iloc[idx]["id"].values
    qzd[f"candidate_{rank+1}_level1"] = targets.iloc[idx]["parent_name"].values
    qzd[f"candidate_{rank+1}_level2"] = targets.iloc[idx]["name"].values

# 6. 简单置信度分层
def confidence(score):
    if score >= 0.78:
        return "可自动匹配"
    if score >= 0.62:
        return "建议人工复核"
    return "无合适匹配"

qzd["match_status"] = qzd["candidate_1_score"].apply(confidence)

qzd.to_excel(out_file, index=False)
print("done:", out_file)
```

这里的关键点是：

- `target_text`：原安美行业，例如 `安美行业：表面处理 / 表面处理涂装`
- `source_text`：企知道行业，例如 `企知道行业：制造业 / 化学原料和化学制品制造业 / 涂料、油墨、颜料及类似产品制造`
- `normalize_embeddings=True` 后，两个向量点积就是余弦相似度
- 不要只看 Top 1，建议输出 Top 3 给人工看
- 阈值 `0.78 / 0.62` 只是初始值，最好拿几十条人工样本校准

例如你给的：

```text
制造业 / 化学原料和化学制品制造业 / 涂料、油墨、颜料及类似产品制造
```

可能会匹配到：

```text

```


原行业表里未必有合适项，这类就应该进入“无合适匹配”或“人工复核”。
