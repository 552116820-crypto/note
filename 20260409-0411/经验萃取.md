这是一场由 **AACTP | 众行集团** 主办的大型会议，主题围绕**组织经验萃取、AI赋能、复盘方法论和人才培养**展开，包含多个演讲主题。

## 一、AI落地与经验萃取结合

**三大误区正在毁掉你的AI项目：**
- **业务层**：AI到底能做多少？经验是否可复制？（能力问题）
- **IT层**：业务是否真正认可？是否愿意使用？（可信问题）
- **决策层**：AI决策是否可靠？是否可审计、可追溯？（落地问题）

**AI落地核心原则：**
1. **场景选择决定成败** -- 适合业务目标清晰、流程明确、可拆解为多步骤的场景；AI作为辅助工具而非最终决策者
2. **不是"独角戏"** -- 需要高层领导、业务中层、信息化部门、基层一线的上下协同
3. **构建可控治理体系** -- 可控原则 + 治理机制 + 评价导向
4. **最佳分工**：80%自动化（AI负责高重复、标准化流程、成本敏感环节） + 20%人类决策（高价值判断、结果背书、异常处理）
5. **三阶段演进**：准备阶段（让AI"能用"）→ 落地阶段（让AI"可控"）→ 运营阶段（让AI"有价值"）

**实战路径：五步构建企业智能体（定-挖-萃-建-优）：**
1. **定** -- 锚定价值场景（选战场、圈地盘、找机会）
2. **挖** -- 挖痛点根因，界定协同机会（阿波罗根因分析）
3. **萃** -- 萃专家经验，构建数字空间（四维对话：技术/用户/商务/风险）
4. **建** -- 搭积木，配人机，落应用
5. **优** -- 优迭代闭环，持续释放价值（数据驱动、用户反馈、专家评审、CI/CD）

**项目成果：** 覆盖15个岗位、1000+个场景、700+个智能体、100W+次调用

## 二、SBP经验萃取工具

一套三步法的结构化经验萃取工具：

1. **Select Case（选择案例）**：从理性维度（富有成效、具体聚焦）和感性维度（感性喜悦、具有挑战）两方面筛选Outstanding Case
2. **Balanced Review（理性复盘）**：用拼图式六维框架 -- 背景Situation、任务Task、行动Action、结果Result、对标Benchmark、反思Review
3. **Perceptual Encapsulation（感性封装）**：四个维度 -- 信息Objective（印象最深刻？）、感受Reflective（感受最强烈？）、启发Interpretive（最有价值的规律？）、行动Decision（最要紧的行动？）

## 三、复盘+萃取方法论

**理论基础：** 大卫·库伯的经验学习圈 -- 积极实践 → 产生经验 → 反思性观察 → 抽象概念化，循环往复。

**核心理念：**
- "不盘不萃不会" -- 任正非：组织最大的浪费就是经验的浪费；AACTP：没有战萃一体的战役就是犯罪
- 复盘是找到真牛人、牛事、牛招及萃取点的捷径
- **边战边萃、战萃一体、战传合一**

**必赢之战中的复盘+萃取时机：**
- 复盘原则：大事阶段盘，关键突发事件马上盘，项目结项全面盘
- 萃取原则：有经验必萃取，有萃取必验证，验证后必固化，固化后必迭代

**124N1团队赋能模式：**
- 1天深度汇谈（目标）→ 2天启动会（计划）→ 4次复盘+萃取（执行）→ N天补差培训 → 1天总复盘（成果）

**AACTP项目复盘画布（Project Review Canvas）：**
- 四大板块：回顾目标（战略解码）→ 评估策略（战术支撑）→ 反思过程（战斗反思）→ 总结规律（战备系统）

## 四、经验萃取核心方法

**淘宝原则（深、透、准）：**
- 对照难痛重点，打破沙锅场域
- 看似相同中找不同，推翻路径依赖
- 不仅挖掘显性行动，更要探询动机假设
- 在偶然中找确定性，多场景举一反三
- 如遇无解疑难杂症，搅拌群智多元共创

**组建淘宝队（5-8人）：**
- 牛人（绩优团队/员工，经验分享）
- 经验使用者（骨干/新人/员工代表，提问挖掘）
- 团队赋能者（管理者/内部专家，激发激励）

**访谈方法：** "访谈是挖矿，研讨是炼金。只挖不炼，矿石变不成金子。" 访谈流程：目标聚焦 → 场景回放 → 提炼步骤 → 追问细节 → 深挖难点 → 复述确认 → 表示感谢

**高意愿驱动（三高中最难但最关键的一环）：**
- 外在驱动：物质奖励、项目奖金、晋升挂钩
- 内在动机：荣誉命名（以名字命名方法论）、专家身份
- 商业化驱动：知识付费、案例收益分享

## 五、人才培养与课程体系

**师课同建关键四步：**
1. **寻源** -- 到炮火一线找到打胜仗的人
2. **铸模** -- 把"个人绝活"提炼成"组织标准"
3. **建课** -- 从"教材思维"到"用户思维"
4. **育师** -- 点燃一个人，照亮一批人

**好课程公式：** 好课程 = 初稿 x（辅导 + 优化 + 试讲 + 反馈）^n

**PECA模型（从"学废"到"学会"的密码）：**
- P - 知识结构化（Programming Knowledge）
- E - 模拟体验（Experience）
- C - 场景化连接（Connecting）
- A - 创设行动（Action）

**训战式岗位经验萃取模式（定-萃-训-战-盘）：** 精准定向(1周) → 经验萃取+AI(2周) → 训练导师(2周) → 实战应用(2-3周) → 复盘迭代(1-2月)

**CSA人才发展经典模型：** 挑战(Challenge) + 支持(Support) + 评估(Assessment)

**赋能内部专家：** 将个人经验打造成"个人知识IP" -- 精讲一门微课/课程、绘制一张地图、留下一套心法

## 六、组织知识管理升级

**SECI模型（组织知识转化与创造）：**
- 社会化(Socialization) → 外显化(Externalization) → 组合化(Combination) → 内隐化(Internalization)

**从"文档库"到"智慧大脑"：**
- 基础工作：知识显性化（建立经验案例库、构建方法论模型库、完善标签体系与检索功能）
- 进阶方向：打造智慧大脑（知识自动关联、AI辅助分析提炼、数据驱动决策）

**进阶阶段（智能化）：** 全域知识图谱 + AI辅助萃取 + 智能推荐引擎

**从"一次性项目"到"可持续的智慧运营"：** 建立常态化"智慧运营"机制，将经验萃取深度融入业务全流程

**文化营造：** 打造"越用越活、越分享越增值"的智慧共同体
