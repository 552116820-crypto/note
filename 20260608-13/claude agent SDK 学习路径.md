Claude Agent SDK 的官方使用文档可以通过以下几种方式访问，已为你整理好：

### 📝 官方文档与资源
*   **英文官方文档**：访问 [`platform.claude.com`](https://platform.claude.com/docs/en/agent-sdk/overview)，可直接获取最新的官方英文使用指南，系统性掌握 SDK 的核心概念与使用方法。
*   **中文官方文档**：访问 [`docs.claude.com`](https://docs.claude.com/zh-CN/api/agent-sdk/overview)，这是官方的中文版本文档，更适合中文开发者阅读。
*   **API 参考**：如果你需要查阅具体的类、方法和参数，完整的英文 API 参考文档同样位于 [`platform.claude.com`](https://platform.claude.com/docs/en/agent-sdk/overview)。
*   **GitHub 仓库**：访问 [`github.com/anthropics`](https://github.com/anthropics)，在这里可以找到 SDK 的**源代码**。同时，官方也提供了丰富的 **Demo 示例**仓库（`github.com/anthropics/claude-agent-sdk-demos`），包含邮件助手、研究 Agent 等完整项目，建议直接 fork 一份。
*   **PyPI 与 npm**：
    *   **Python**：`claude-agent-sdk` 包页面 [pypi.org/project/claude-agent-sdk](https://pypi.org/project/claude-agent-sdk) 提供了Python 包的安装说明和简要文档。
    *   **TypeScript/JavaScript**：`@anthropic-ai/claude-agent-sdk` 包页面 [www.npmjs.com/package/@anthropic-ai/claude-agent-sdk](https://www.npmjs.com/package/@anthropic-ai/claude-agent-sdk) 提供了TypeScript 包的安装说明。
*   **社区教程与文章**：中文社区有许多优质的教程资源可以作为补充学习材料：
    *   **CSDN**：有系列实战指南，如《Claude Agent SDK 深度入门指南》和《实战：用 Python 构建 AI Agent》。
    *   **腾讯云/百度智能云**：提供了从零到一的开发教程。
    *   **开发者社区**：掘金、GitCode 等平台上有快速开始和全面的实战指南。
    *   **微信公众号**：搜索“Claude Agent SDK”可以找到一些中文的使用介绍和心得分享。
*   **开发工具**：有第三方平台（如 Portkey）提供了将 Claude Agent SDK 用于生产环境的指南，包括可观测性、预算限制等功能。Microsoft Learn 上也有关于在特定环境中设置和运行 JavaScript Agent 的快速入门教程。
