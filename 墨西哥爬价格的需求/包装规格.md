• Drum 和 Pail 主要是包装大小不同，不是油品等级不同。

  - Drum：大桶，通常是铁桶/工业桶。润滑油里常见是 55 gal，约 208-209L；润滑脂常见是 170-180kg。
  - Pail：小桶、提桶、方桶。润滑油里常见是 5 gal，约 18.9-20L；润滑脂常见是 16-18kg。

  实际差别通常是：

  - Drum 单位价格通常更低
  - Pail 更方便搬运、分装和小批量使用
  - 物流、仓储、取用方式不同

  在你这些润滑油表里，可以简单理解成：

  - 液体产品：Drum 往往对应大包装，Pail 对应小包装
  - 润滑脂：Drum 往往是大重量桶，Pail 是小重量桶

  所以表里的 Drum 和 Pail 价格不能直接比总价，最好比 每升 或 每公斤。另外，不同品牌的 Drum/Pail 容量不一定完全一样，最终还是要看具体包装字段，比如 209L、19L、18kg。


• IBC 和 Bulk 也都是包装/供货方式，通常比 drum 更大。

  - IBC
      - Intermediate Bulk Container
      - 中型散装容器，常见是带金属框架的塑料吨桶
      - 液体润滑油常见容量是 1000L 左右
      - 适合中大批量采购，搬运比散装方便，比小桶更省包装成本
  - Bulk
      - 散装、散液
      - 通常不是固定小包装，而是槽车、储罐、管线灌装供货
      - 一般按实际提货量结算
      - 适合很大批量、连续用油场景

  可以简单理解为包装规模从小到大大致是：

  Pail < Drum < IBC < Bulk

  区别主要在：

  - IBC：有容器，通常是一整只吨桶
  - Bulk：通常没标准零售包装，直接按散装交付

  所以比价时也一样，不看总价，要看 每升 或 每公斤。
