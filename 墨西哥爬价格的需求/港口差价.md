• 因为这份 Shell PDF 里的 LIST PRICES 不是“墨西哥到港最终价”，而是 全球基础价。
  如果你要算 墨西哥港口参考价，就必须再加 Mexico port price differential。

  逻辑是这样：

  - 第 7-10 页 的 LIST PRICES
      - 是产品基础价
      - 单位是 USD/100L 或 USD/100kg
  - 第 4 页 和 第 11-16 页
      - 说明了 port price differential 是按供货港口加收的差价
      - 不同港口差价不同
  - 第 16 页
      - 列了 Mexico 各港口差价，像 60、70

  所以：

  墨西哥港口参考价 = 产品基础价 + 墨西哥港口差价

  例如 Shell Tellus S2 MX 32：

  - 基础价：
      - Drum = 923
      - Pail = 963
  - 墨西哥港口差价：
      - 60 到 70

  所以才会变成：

  - Drum: (923+60)/100 = 9.83
  - Drum: (923+70)/100 = 9.93
  - Pail: (963+60)/100 = 10.23
  - Pail: (963+70)/100 = 10.33

  也就是说，你表里那个区间不是随便加的，而是因为：

  - 墨西哥不是一个统一港口价
  - 不同墨西哥港口差价不同
  - 所以只能写成区间

  补一句：这个 PDF 本质上是 Shell Marine 的国际港口供货价，不是墨西哥本地电商零售价。
  如果你要的是“墨西哥官方港口参考价”，要加港口差价；如果你要的是“全球基础挂牌价”，就不加。

在这个 PDF 里，墨西哥价格 不是单独一页成品价，而是要用 产品基价 + Mexico 港口差价 换算出来。

  文件：Downloads/shell-marine-price-guide-april-2022.pdf

  关键位置：

  - 第 7-10 页：LIST PRICES
      - 这里是各产品的基础价
      - 口径写得很清楚：Product prices in US dollars per: BULK / IBC / DRUM / PAILS AND SMALL PACKAGES
      - 单位是 USD per 100 L 或 USD per 100 kg
  - 第 16 页：PORT PRICE DIFFERENTIALS
      - 这里有 MEXICO
      - 墨西哥各港口差价：
          - ALTAMIRA 60
          - CIUDAD DEL CARMEN 70
          - COATZACOALCOS 70
          - COZUMEL 70
          - DOS BOCAS 70
          - ENSENADA 70
          - LAZARO CARDENAS 60
          - MANZANILLO (MX) 70
          - MAZATLAN 60
          - PROGRESO 70
          - SALINA CRUZ 60
          - TAMPICO 70
          - TUXPAN 70
          - VERACRUZ 60

  换算逻辑就是：

  墨西哥港口参考价(USD/100L 或 USD/100kg)
  = LIST PRICES 对应包装基础价 + Mexico port differential

  再换成 USD/L 或 USD/kg：
  = 上述结果 ÷ 100

  例子：

  - Shell Tellus S2 MX 32 在 第 7 页
      - DRUM = 923
      - PAIL = 963
  - Mexico 港口差价在 第 16 页
      - 最低 60
      - 最高 70

  所以：

  - Drum
      - (923 + 60) / 100 = 9.83 USD/L
      - (923 + 70) / 100 = 9.93 USD/L
  - Pail
      - (963 + 60) / 100 = 10.23 USD/L
      - (963 + 70) / 100 = 10.33 USD/L

  这就是你表里那种 9.83-9.93、10.23-10.33 的来源。
