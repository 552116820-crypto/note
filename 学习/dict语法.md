seen={}
• 这里的原理其实就是 Python 字典 dict 的赋值语法。

  seen[num] = i

  意思是：

  - 把 num 当作 键 key
  - 把 i 当作 值 value
  - 存进字典 seen 里

  例如：

  seen = {}
  seen[2] = 0
  seen[7] = 1

  此时 seen 就变成：

  {2: 0, 7: 1}

  也就是：

  - 数字 2 对应下标 0
  - 数字 7 对应下标 1

  底层上，dict 本质是一个 哈希表：

  1. Python 先对键 num 计算哈希值
  2. 根据哈希值找到存储位置
  3. 把这个键和值保存进去

  所以后面你写：

  if need in seen:

  或者：

  seen[need]

  都能很快找到对应的下标。

  在 Two Sum 里，seen[num] = i 的作用就是：
  “把当前数字和它的位置记下来，方便后面别的数字来找它配对。”
    如果同一个数字重复出现，比如：

  seen[2] = 0
  seen[2] = 3

  那么后一次会把前一次覆盖掉，结果变成：

  {2: 3}
