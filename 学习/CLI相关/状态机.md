• 状态机就按 状态 + 事件 + 守卫(校验) + 持久化 来做，先做最小可运行版，不要一开始上复杂框架。  

最小状态机设计


  1. PLANNED：Claude 产出任务卡。
  2. READY：任务卡通过 schema 校验。
  3. IMPLEMENTING：Codex 正在实现。
  4. REVIEWING：CI 通过，等待 Claude 评审。
  5. REWORK：评审或CI失败，回修。
  6. DONE：通过并可合并。        
  7. BLOCKED：被外部依赖阻塞。   
  8. CANCELED：任务取消。        

  事件定义
  1. VALIDATE_OK
  2. START_IMPL
  3. CI_OK
  4. CI_FAIL
  5. REVIEW_PASS
  6. REVIEW_FAIL
  7. BLOCK
  8. UNBLOCK
  9. CANCEL

 "READY": { "START_IMPL": "IMPLEMENTING", "BLOCK": "BLOCKED", "CANCEL": "CANCELED" },

  PowerShell 核心函数（可直接用）
 
  function Send-Event {
    param(
      [string]$TaskId,
      [string]$Event,
      [string]$Actor = "orchestrator",
      [string]$Note = ""
    )
    $statePath = "runs/$TaskId.state.json"
    $transPath = "pipeline/transitions.json"
    $s = Get-Content $statePath -Raw | ConvertFrom-Json
    $t = Get-Content $transPath -Raw | ConvertFrom-Json
 
    $from = [string]$s.state
    $row = $t.$from
    if (-not $row) { throw "Unknown state: $from" }
 
    $to = $row.$Event
    if (-not $to) { throw "Invalid transition: $from --$Event--> ?" }
    # 守卫校验：按 from+event 做硬约束
    switch ("$from|$Event") {
      "PLANNED|VALIDATE_OK" { Test-TaskSchema $TaskId | Out-Null }
      "READY|START_IMPL"    { Assert-BaseCommit $TaskId }
      "IMPLEMENTING|CI_OK"  { Assert-CIPassed $TaskId; Assert-AllowFiles $TaskId }
      "REVIEWING|REVIEW_PASS" { Assert-ReviewExists $TaskId }
    }
 
    if ($Event -eq "BLOCK") { $s.blocked_from = $from }
    if ($from -eq "BLOCKED" -and $Event -eq "UNBLOCK") { $to = $s.blocked_from }
 
    $s.state = $to
    $s.version = [int]$s.version + 1
    $entry = [pscustomobject]@{
      at    = (Get-Date).ToString("s")
      event = $Event
      from  = $from
      to    = $to
      actor = $Actor
      note  = $Note
    }
    $s.history += $entry
 
    $s | ConvertTo-Json -Depth 10 | Set-Content $statePath -Encoding UTF8
  }

  编排循环（关键）
  1. 当前状态是 READY 时，触发 Codex 实现，然后发 START_IMPL。
  2. Codex提交后跑 CI，成功发 CI_OK，失败发 CI_FAIL。
  3. REVIEWING 时触发 Claude 审查，过审发 REVIEW_PASS，否则 REVIEW_FAIL。
  4. 到 DONE 自动创建合并动作；REWORK 回到 Codex。
 
  对接问题怎么靠状态机解决
 
  1. 模型输出不稳定：只接受事件，不接受“直接改状态”。
  2. 越界修改：IMPLEMENTING -> CI_OK 前强制 Assert-AllowFiles。
  3. 标准不一致：守卫只认任务卡 acceptance。
  4. 并发冲突：每个任务独立 state.json + 文件锁。

  建议

  1. 用 Python 写 orchestrator（调 Claude/Codex、跑 CI、推进状态）。
  2. 用 JSON 文件持久化状态（每个任务一个 runs/T-xxx.state.json）。
  3. 用 jsonschema 做任务卡/结果卡校验，用 filelock 防并发写冲突。 


```python
  # orchestrator.py
  from __future__ import annotations
  import argparse
  import json
  from dataclasses import dataclass, field, asdict
  from datetime import datetime, timezone
  from pathlib import Path
  from typing import Optional

  from filelock import FileLock

  TRANSITIONS = {
      "PLANNED": {"VALIDATE_OK": "READY", "BLOCK": "BLOCKED", "CANCEL": "CANCELED"},
      "READY": {"START_IMPL": "IMPLEMENTING", "BLOCK": "BLOCKED", "CANCEL": "CANCELED"}, 
      "IMPLEMENTING": {"CI_OK": "REVIEWING", "CI_FAIL": "REWORK", "BLOCK": "BLOCKED"},   
      "REVIEWING": {"REVIEW_PASS": "DONE", "REVIEW_FAIL": "REWORK", "BLOCK": "BLOCKED"}, 
      "REWORK": {"START_IMPL": "IMPLEMENTING", "BLOCK": "BLOCKED", "CANCEL": "CANCELED"},
      "BLOCKED": {"UNBLOCK": "__PREV__", "CANCEL": "CANCELED"},
      "DONE": {},
      "CANCELED": {},
  }

  @dataclass
  class TaskState:
      task_id: str
      state: str = "PLANNED"
      blocked_from: Optional[str] = None
      version: int = 1
      history: list[dict] = field(default_factory=list)
 
  def now_iso() -> str:
      return datetime.now(timezone.utc).isoformat()    
 
  def state_path(task_id: str) -> Path:
      p = Path("runs")
      p.mkdir(parents=True, exist_ok=True)
      return p / f"{task_id}.state.json"
 
  def load_state(task_id: str) -> TaskState:
      p = state_path(task_id)
      if not p.exists():
          return TaskState(task_id=task_id)
      data = json.loads(p.read_text(encoding="utf-8"))
      return TaskState(**data)
 
  def save_state(s: TaskState) -> None:
      p = state_path(s.task_id)
      p.write_text(json.dumps(asdict(s), ensure_ascii=False, indent=2), encoding="utf-8")
 
  def guard(from_state: str, event: str, task_id: str) -> None:
      # 这里放硬校验逻辑（示例留空）:
      # - PLANNED + VALIDATE_OK: 校验 task.json schema
      # - IMPLEMENTING + CI_OK: 校验 CI 结果、allow_files
      # - REVIEWING + REVIEW_PASS: 校验 review 文件存在且 pass
      pass

  def send_event(task_id: str, event: str, actor: str = "orchestrator", note: str = "") -> TaskState:
      lock = FileLock(str(state_path(task_id)) + ".lock")
      with lock:
          s = load_state(task_id)
          from_state = s.state
          row = TRANSITIONS.get(from_state, {})
          if event not in row:
              raise ValueError(f"Invalid transition: {from_state} --{event}--> ?")

          guard(from_state, event, task_id)

          to_state = row[event]
          if event == "BLOCK":
              s.blocked_from = from_state
          if from_state == "BLOCKED" and event == "UNBLOCK":
              to_state = s.blocked_from or "READY"
 
          s.state = to_state
          s.version += 1
          s.history.append({
              "at": now_iso(),
              "event": event,
              "from": from_state,
              "to": to_state,
              "actor": actor,
              "note": note
          })
          save_state(s)
          return s
 
  def main():
      ap = argparse.ArgumentParser()
      ap.add_argument("--task", required=True, help="e.g. T-012")       
      ap.add_argument("--event", required=True, help="e.g. VALIDATE_OK")
      ap.add_argument("--actor", default="orchestrator")
      ap.add_argument("--note", default="")
      args = ap.parse_args()
 
      s = send_event(args.task, args.event, args.actor, args.note)      
      print(json.dumps(asdict(s), ensure_ascii=False, indent=2))        
  if __name__ == "__main__":
      main()
```

  接入位置

  1. PLANNED -> READY 前：调用 Claude 生成/校验任务计划。
  2. READY/REWORK -> IMPLEMENTING：调用 Codex 产出代码变更（通常返回 patch 或文件改动）。
  3. IMPLEMENTING -> REVIEWING：本地跑 CI。
  4. REVIEWING -> DONE/REWORK：调用 Claude 做代码评审并给出 pass/fail。

  下面是可直接套用的 Python 骨架（把你现有状态机函数复用进去）：

```python
  import os
  import json
  import subprocess
  from pathlib import Path
 
  from openai import OpenAI
  from anthropic import Anthropic

  # 你已有的状态机函数
  # - load_state(task_id)
  # - send_event(task_id, event, actor, note="")

  openai_client = OpenAI(api_key=os.environ["OPENAI_API_KEY"])
  claude_client = Anthropic(api_key=os.environ["ANTHROPIC_API_KEY"])
 
  CLAUDE_MODEL = os.getenv("CLAUDE_MODEL", "claude-sonnet-4-5")
  CODEX_MODEL = os.getenv("CODEX_MODEL", "gpt-5")  # 这里放你可用的Codex/代码模型
  def load_task(task_id: str) -> dict:
      return json.loads(Path(f"tasks/{task_id}.task.json").read_text(encoding="utf-8"))

  def save_json(path: str, data: dict):
      Path(path).parent.mkdir(parents=True, exist_ok=True)
      Path(path).write_text(json.dumps(data, ensure_ascii=False, indent=2), encoding="utf-8")
 
  def call_claude_plan(task: dict) -> dict:
      prompt = f"""你是项目规划器。基于任务卡输出严格JSON:
  {{
    "task_id": "{task['task_id']}",
    "plan": ["..."],
    "risks": ["..."],
    "acceptance_checklist": ["..."]
  }}
  任务卡:
  {json.dumps(task, ensure_ascii=False)}
  只输出JSON。
  """
      resp = claude_client.messages.create(
          model=CLAUDE_MODEL,
          max_tokens=2000,
          temperature=0.2,
          messages=[{"role": "user", "content": prompt}],
      )
      text = "".join(
          block.text for block in resp.content
          if getattr(block, "type", "") == "text"
      )
      return json.loads(text)

  def call_codex_generate_patch(task: dict, plan: dict) -> str:
      prompt = f"""你是代码实现代理。根据任务与计划生成 unified diff patch（git apply 可用）。
  只输出patch内容，不要解释。
 
  任务:
  {json.dumps(task, ensure_ascii=False)}

  计划:
  {json.dumps(plan, ensure_ascii=False)}
  """
      resp = openai_client.responses.create(
          model=CODEX_MODEL,
          input=prompt,
          temperature=0.1,
      )
      return resp.output_text
 
  def apply_patch(patch_text: str, task_id: str):
      patch_file = Path(f"runs/{task_id}.patch")
      patch_file.parent.mkdir(parents=True, exist_ok=True)
      patch_file.write_text(patch_text, encoding="utf-8")
      subprocess.run(["git", "apply", str(patch_file)], check=True)
 
  def run_ci(task: dict) -> bool:
      cmds = task.get("commands", {})
      for key in ("lint", "test"):
          cmd = cmds.get(key)
          if not cmd:
              continue
          result = subprocess.run(cmd, shell=True)
          if result.returncode != 0:
              return False
      return True
 
  def call_claude_review(task: dict) -> dict:
      diff = subprocess.check_output(["git", "diff", "--", *task.get("allow_files", [])], text=True)
      prompt = f"""你是代码评审。按任务验收标准给出严格JSON:
  {{
    "task_id": "{task['task_id']}",
    "decision": "pass" | "fail",
    "findings": [{{"severity":"high|medium|low","file":"...","note":"..."}}]
  }}
  验收标准:
  {json.dumps(task.get("acceptance", []), ensure_ascii=False)}
  代码diff:
  {diff[:12000]}
  只输出JSON。
  """
      resp = claude_client.messages.create(
          model=CLAUDE_MODEL,
          max_tokens=2000,
          temperature=0.1,
          messages=[{"role": "user", "content": prompt}],
      )
      text = "".join(
          block.text for block in resp.content
          if getattr(block, "type", "") == "text"
      )
      return json.loads(text)
 
  def process_task(task_id: str):
      task = load_task(task_id)
      state = load_state(task_id).state
 
      if state == "PLANNED":
          plan = call_claude_plan(task)
         save_json(f"handoff/{task_id}.plan.json", plan)
          send_event(task_id, "VALIDATE_OK", "claude")
 
      state = load_state(task_id).state
      if state in ("READY", "REWORK"):
          send_event(task_id, "START_IMPL", "orchestrator")
          plan = json.loads(Path(f"handoff/{task_id}.plan.json").read_text(encoding="utf-8"))
          patch = call_codex_generate_patch(task, plan)
          apply_patch(patch, task_id)
          ci_ok = run_ci(task)
          send_event(task_id, "CI_OK" if ci_ok else "CI_FAIL", "ci")
          if not ci_ok:
              return
       state = load_state(task_id).state
      if state == "REVIEWING":
          review = call_claude_review(task)
          save_json(f"reviews/{task_id}.review.json", review)
          send_event(task_id, "REVIEW_PASS" if review["decision"] == "pass" else "REVIEW_FAIL", "claude")
```
