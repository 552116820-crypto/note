**Assistant:**

**套磁 = 主动给目标导师发邮件，让他在录取前认识你、看上你。**

"套"取自"套近乎"，"磁"取自"磁铁"——把导师"吸引"过来。是中国研究生申请圈的黑话，海归博士申请也用这个词。

## 它到底起什么作用

考研有"初试 + 复试"两关。**复试时导师有相当大的话语权**，能在差不多的考生里挑自己中意的。套磁的核心目的，就是让你在导师那里**从"复试时见过的众多陌生人"变成"已经认识、有印象的人"**。

对你这种情况，套磁的杠杆**特别大**：
- 你的劣势（双非本科 + 跨考食品）写在档案上一眼可见，无可掩饰
- 你的优势（3 年 AI 工程经验、做过 Agent + RAG）必须**主动让导师知道**，否则纸面信息没人看得出
- 复试面试官就是目标导师，提前混个脸熟可以直接改变面试体验

## 套磁的关键时间窗

| 时间 | 适合做什么 |
|---|---|
| **6–8 月**（推免开始前）| 第一波套磁，主要试探导师当年招生计划 |
| **9–10 月**（推免季） | 不建议——导师在忙保研生面试，没空看陌生邮件 |
| **12 月**（初试） | 闭嘴备考 |
| **2 月**（出分后）| **黄金期**。带着分数套磁，导师会认真看 |
| **3–4 月**（复试前）| 最后一波，确认导师意向 |

你现在是 2026 年 5 月，**6 月就是第一波合适的时间点**。

## 一封"有效的"套磁邮件长什么样

它不是简历群发，是**针对单个导师量身写的 200–400 字邮件**，结构通常是：

1. **一句自我介绍**（学校+届次+目标）
2. **为什么是这个导师**（你读过他哪篇 paper、对哪个方向感兴趣，**必须具体到论文标题**，否则一眼看出群发）
3. **你的"硬通货"**（3 年 AI 工程经验 + 项目 + 开源 / 论文 / 比赛，挑最相关的 2–3 条）
4. **明确的请求**（请问您 2027 年是否招收硕士研究生？我希望能有机会进一步交流）
5. **附件**：简历 PDF + 项目链接

## 套磁的雷区

- **群发 / 模板感重** → 直接被删
- **同一个所给多个导师套磁但没说明** → 圈子小，导师之间会聊，会扣分
- **没读过导师 paper 就吹捧"久仰大名"** → 一句话就被识破
- **简历夸大或造假** → 复试当场翻车
- **写完没下文** → 套磁后要持续跟进（如果导师回复正面，要在初试后第一时间汇报分数）
