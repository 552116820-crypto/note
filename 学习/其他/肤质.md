• 可以自己先做个 30分钟观察法，很实用。

  晚上或早上都行，按这个流程：

  - 用温和洁面洗脸
  - 擦干后 30分钟 内什么都别涂
  - 在自然光下看脸，顺便感受皮肤状态

  这样判断：

  - 油皮：全脸都容易发亮、摸起来有油感，尤其额头鼻子更明显。
  - 干皮：洗完后容易紧绷、粗糙、起皮，甚至痒。
  - 混合皮：T区油，脸颊干，这是最常见的一种。
  - 敏感肌：重点不是油不油，而是你是否经常一用产品就 刺、烧、痒、红。
    也就是说，你可以是 油皮 + 敏感肌，也可以是 干皮 + 敏感肌。

  一个更简单的速判：

  - 发亮多 -> 偏油
  - 紧绷起皮 -> 偏干
  - 又油又容易刺痛泛红 -> 油敏
  - 又干又容易刺痛泛红 -> 干敏

  再提醒你两点：

  - 肤质会变。夏天更油、冬天更干，熬夜、压力、经期、怀孕、用药都会影响。
  - 如果你分不清“干”还是“敏感”，看有没有 刺痛/灼热/明显泛红。单纯干更多是紧绷、粗糙、脱皮；敏感更强调“反应性”。

  你也可以顺手做个 新产品试用测试：

  - 先涂在耳后、下颌线或手臂内侧一小块
  - 连续 7到10天
  - 如果出现红、痒、肿，就别上脸


• 按这个自测，洗脸后等30分钟，什么都别涂，然后回答我每题 是/否：

  1. 脸会明显紧绷、粗糙，甚至有点起皮吗？
  2. 全脸都容易发亮、摸起来有油感吗？
  3. 只有 T区（额头、鼻子） 明显出油，但脸颊还好甚至偏干吗？
  4. 你一用新产品，常会刺痛、发红、发痒或发烫吗？
  5. 换季时，你的脸很容易泛红、脱皮或不舒服吗？

  你也可以先自己对照：

  - 1是，2否：偏干皮
  - 2是，1否：偏油皮
  - 3是：混合皮
  - 4或5是：说明有敏感倾向
  - 2是 + 4/5是：油敏肌
  - 1是 + 4/5是：干敏肌

  直接按这个格式回我就行：

  1否 2是 3是 4否 5否
