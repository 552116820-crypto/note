• 我是按“先停用，再卸载，再清残留”的方法做的，不是只删一个程序目录。

  具体做了这些事：

  - 先定位真实安装位置和组件
    找到它不在常规 Program Files，而是在 C:\Inetpub\ftproot\Tipray\LdTerm。
    同时识别出它的关键组件：
      - 服务：LeaderTerm、LeaderTermDaemon
      - 驱动：LdMFilter、Ldcore、ldwfp、LdCdRomFilters
      - 进程：LdTerm.exe、LdApproval.exe、LdFileGate.exe、LdMsgInject*.exe、LdTermPlug*.exe
  - 先禁用自启动和相关服务/驱动
    用管理员命令把这些服务和驱动改成 disabled，防止重启后继续拉起。
  - 杀掉用户态进程
    结束 LdTerm.exe、LdApproval.exe 等前台和注入进程，避免它们占文件、反复拉起。
  - 尝试走它自己的卸载入口
    从安装目录里的 Setup.exe 提取出卸载参数，尝试过：
      - -uninstall
      - -UnInstall
      - /u /s
      - -remove -silent
  - 重启后做强制清理
    因为还有过滤驱动和残留注册，所以重启后继续：
      - 删除服务/驱动注册项
      - 删除 HKLM\SYSTEM\CurrentControlSet\Services\ldwfp
      - 删除残留目录 C:\Inetpub\ftproot\Tipray
      - 删除残留驱动文件：
          - C:\Windows\System32\Drivers\LdMFilter.sys
          - C:\Windows\SysWOW64\Drivers\LDMFilter.sys
          - C:\Windows\SysWOW64\Drivers\LdCdRomFilters.sys
  - 最后复核
    我确认了：
      - 没有绿盾相关进程在跑
      - 相关服务/驱动全部变成“未安装”
      - 安装目录和驱动文件都不存在了

  所以本质上是“组件识别 + 停用 + 官方卸载尝试 + 手工清残留”的组合方案。



• 关键命令大致就是这些，按顺序来的。

  先定位组件：

  sc query LeaderTerm
  sc query LeaderTermDaemon
  sc query Ldcore
  sc query ldwfp
  sc query LdCdRomFilters
  sc query LdMFilter

  tasklist | findstr /i "LdTerm LdApproval LdFileGate LdMsgInject LdTermPlug LdTermDaemon"

  先禁用自启动：

  sc config LeaderTerm start= disabled
  sc config LeaderTermDaemon start= disabled
  sc config Ldcore start= disabled
  sc config ldwfp start= disabled
  sc config LdCdRomFilters start= disabled
  sc config LdMFilter start= disabled

  结束用户态进程：

  taskkill /F /T /IM LdTerm.exe
  taskkill /F /T /IM LdApproval.exe
  taskkill /F /T /IM LdTermPlug.exe
  taskkill /F /T /IM LdTermPlug64.exe
  taskkill /F /T /IM LdMsgInject.exe
  taskkill /F /T /IM LdMsgInject64.exe
  taskkill /F /T /IM LdFileGate.exe
  taskkill /F /T /IM LdTermDaemon.exe

  尝试官方卸载：

  cd /d C:\Inetpub\ftproot\Tipray\LdTerm
  Setup.exe -uninstall
  Setup.exe -UnInstall
  Setup.exe /u /s
  Setup.exe -remove -silent

  强制删服务/驱动注册：

  sc delete LeaderTerm
  sc delete LeaderTermDaemon
  sc delete Ldcore
  sc delete ldwfp
  sc delete LdCdRomFilters
  sc delete LdMFilter

  reg delete "HKLM\SYSTEM\CurrentControlSet\Services\ldwfp" /f

  拿权限并删目录/驱动文件：

  takeown /f C:\Inetpub\ftproot\Tipray /r /d y
  icacls C:\Inetpub\ftproot\Tipray /grant Administrators:F /t
  rmdir /s /q C:\Inetpub\ftproot\Tipray

  takeown /f C:\Windows\System32\Drivers\LdMFilter.sys
  icacls C:\Windows\System32\Drivers\LdMFilter.sys /grant Administrators:F
  del /f /q C:\Windows\System32\Drivers\LdMFilter.sys

  takeown /f C:\Windows\SysWOW64\Drivers\LDMFilter.sys
  icacls C:\Windows\SysWOW64\Drivers\LDMFilter.sys /grant Administrators:F
  del /f /q C:\Windows\SysWOW64\Drivers\LDMFilter.sys

  takeown /f C:\Windows\SysWOW64\Drivers\LdCdRomFilters.sys
  icacls C:\Windows\SysWOW64\Drivers\LdCdRomFilters.sys /grant Administrators:F
  del /f /q C:\Windows\SysWOW64\Drivers\LdCdRomFilters.sys

  最后复核：

  sc query LeaderTerm
  sc query LeaderTermDaemon
  sc query Ldcore
  sc query ldwfp
  sc query LdCdRomFilters
  sc query LdMFilter

  tasklist | findstr /i "LdTerm LdApproval LdFileGate LdMsgInject LdTermPlug LdTermDaemon"
