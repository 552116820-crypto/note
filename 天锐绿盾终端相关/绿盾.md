• 下面这套按“先停用，再走官方卸载，最后才强拆”的顺序来。已确认这台机器上的相关组件是：

  - 服务：LeaderTerm、LeaderTermDaemon
  - 驱动：Ldcore、ldwfp、LdCdRomFilters、LdMFilter
  - 安装目录：/mnt/c/inetpub/ftproot/Tipray/LdTerm
  - 安装器：/mnt/c/inetpub/ftproot/Tipray/LdTerm/Setup.exe

  1. 先在“管理员 CMD”里停用并尽量停掉它

  cd /d C:\Inetpub\ftproot\Tipray\LdTerm

  sc config LeaderTerm start= disabled
  sc config LeaderTermDaemon start= disabled
  sc config Ldcore start= disabled
  sc config ldwfp start= disabled
  sc config LdCdRomFilters start= disabled
  sc config LdMFilter start= disabled

  taskkill /F /T /IM LdTerm.exe
  taskkill /F /T /IM LdApproval.exe
  taskkill /F /T /IM LdTermPlug.exe
  taskkill /F /T /IM LdTermPlug64.exe
  taskkill /F /T /IM LdMsgInject.exe
  taskkill /F /T /IM LdMsgInject64.exe
  taskkill /F /T /IM LdTermDaemon.exe

  sc stop LeaderTerm
  sc stop LeaderTermDaemon
  sc stop Ldcore
  sc stop ldwfp
  sc stop LdCdRomFilters
  fltmc unload LdMFilter

  LdMFilter 是启动级过滤驱动，fltmc unload LdMFilter 如果失败很常见，说明它需要重启后才会真正卸下。

  2. 优先尝试官方卸载

  在同一个“管理员 CMD”里，一条一条试，命中一个就停，不要四条一起跑：

  cd /d C:\Inetpub\ftproot\Tipray\LdTerm
  start /wait "" Setup.exe -uninstall

  如果上面没反应，再试：

  start /wait "" Setup.exe -UnInstall
  start /wait "" Setup.exe /u /s
  start /wait "" Setup.exe -remove -silent

  Setup.exe 里有 UninstallCode 相关逻辑，所以如果弹窗要“卸载码”，那就是它的正式卸载流程，需要找 IT 要码。

  3. 如果官方卸载走不通，再做强制清理

  先重启一次，再进“管理员 CMD”执行：

  sc delete LeaderTerm
  sc delete LeaderTermDaemon
  sc delete Ldcore
  sc delete ldwfp
  sc delete LdCdRomFilters
  sc delete LdMFilter

  然后清理目录权限并改名，避免它被再次拉起：

  takeown /f C:\Inetpub\ftproot\Tipray /r /d y
  icacls C:\Inetpub\ftproot\Tipray /grant Administrators:F /t
  ren C:\Inetpub\ftproot\Tipray Tipray.disabled

  最后重启：

  shutdown /r /t 0

  sc query ldwfp
