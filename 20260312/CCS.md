• CSS 是 Cascading Style Sheets，中文通常叫“层叠样式表”。

  它的作用是控制网页的外观，比如：

  - 文字颜色
  - 字体大小
  - 间距
  - 边框
  - 背景
  - 布局
  - 动画

  简单理解：

  - HTML 负责内容和结构
  - CSS 负责样式和排版
  - JavaScript 负责交互和行为

  例如：

  <p class="title">你好</p>

  .title {
    color: blue;
    font-size: 24px;
  }

  这段 CSS 的意思是把这段文字设为蓝色、24 像素。
